## PROJETO

Neste projeto está sendo desenvolvido um site para votação e apuração dos votos do reality de entretenimento online **"A Granja"**!

## EXECUTANDO

- Clone ou Baixe este repositório

- Abra-o no terminal

- Execute:
```bash
npm install
```
```bash
npm start
```
